# Static-Routing

The project shows how to configure static routing between different networks with the help of Cisco routers on Packet Tracer. It covers IP addressing, subnet configuration, default gateways and router configurations to allow end-to-end connectivity and to verify successful communication between subnets.

Components Used:
Cisco Routers
Cisco Switches
PCs (End Devices)
Ethernet Copper Straight-Through Cables
Cisco Packet Tracer

Project Outcome:
Successfully configured and verified static routing between multiple networks in Cisco Packet Tracer. Assigned IP addresses, configured default gateways and static routes, and achieved end-to-end connectivity across all subnets. Connectivity was validated using ping tests and Cisco IOS verification commands.
